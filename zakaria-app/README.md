# Employee ID Card Template  

A clean and responsive employee ID card built with HTML & CSS. Easily customizable for company use.

## Features  
- Modern, professional design  
- Responsive layout (mobile & desktop)  
- Easy to edit: replace logo, name, photo, details  
- Print-ready styling  

## How to Use  
1. Open index.html in your browser.  
2. Edit the data in the file (name, position, ID, etc.).  
3. Replace logo.png with your company logo.  

## Technologies  
HTML5, CSS3, (Flexbox)
